# Community Discussion Forum with Real-Time Chat

Minimal MERN + Socket.IO example built for learning and demonstration.

Quickstart

Server
```
cd server
npm install
cp .env.example .env
# edit .env
npm run dev
```

Client
```
cd client
npm install
# set REACT_APP_API_URL and REACT_APP_SOCKET_URL
npm run dev
```

Don't commit `.env`. Add `.env.example` instead.
